# AdofaiMacro

A Dance of Fire and Ice (ADOFAI) 游戏的宏工具。

## 系统要求

- Java 21 或更高版本
- 支持 JavaFX 的系统环境

## 构建说明

本项目使用 Gradle 构建系统。要构建项目：

```bash
./gradlew build
```

要运行项目：

```bash
./gradlew run
```

要创建独立的可分发包：

```bash
./gradlew jlinkZip
```

## 开发环境设置

1. 确保安装了 Java 21 JDK
2. 克隆此仓库
3. 使用支持 Gradle 的 IDE（如 IntelliJ IDEA）打开项目
4. 等待 Gradle 同步完成

## 依赖项

- JavaFX 23.0.1
- json-simple 1.1.1
- JNativeHook 2.1.0
